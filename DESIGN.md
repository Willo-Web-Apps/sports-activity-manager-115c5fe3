# Design DNA — Sideline

## Visual Personality: organic-natural
## Layout: product-led
## Typography: classic-balanced

### Color Palette
- Primary: #2E6B4F (forest green — reliable, grounded)
- Accent: #E8925A (warm orange — energy, warmth, youth)
- Surface: #FAF6F1 (warm off-white — soft, inviting, clean)
- Text: oklch(0.22 0.03 145) (deep forest, warm dark)

### Font Pairing
- Display: IBM Plex Sans — 400/600/700 (professional, approachable, warm)
- Body: IBM Plex Serif — 400/600 (trustworthy, readable, distinct from competitors)

### Unique Design Choices
- Warm off-white (#FAF6F1) base surface — immediately differentiates from cold-white competitors (TeamSnap, Cozi)
- Dashboard "product visual" built from CSS — colored cards with soft shadows, mimicking a real family dashboard UI
- Asymmetric hero layout: headline/CTA left, product visual floats right — breaks from the centered hero norm
- Forest green + warm orange gradient orbs for hero background — feels like morning light through trees, not corporate SaaS
- "Works for every activity" section uses organic pill-tag grid instead of standard icon feature cards
- Noise texture overlay on hero for hand-crafted feel vs. AI/corporate gloss
- Green-tinted section for mid-page break uses a subtly textured background, not a flat color block
